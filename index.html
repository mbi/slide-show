<slide-show>

The accessible, scrollable, styleable, horizontal carousel custom element. No dependencies, about 12kB minified and gzipped.

A <slide-show> element treats its children as slides, layed out in a grid by default. Slide content may be any arbitrary HTML.

API

Attributes

autoplay

Boolean attribute. When present the slide-show activates the next slide after a pause. The pause duration may be set in CSS via the --slide-duration variable.

controls

A TokenList attribute. Accepts the tokens "navigation", "pagination" and "fullscreen".

loop

Boolean attribute. Makes the slideshow behave as a continuous loop.

Properties

.active

Gets or sets the currently scroll-snapped child element.

const activeSlide = slideshow.active;

May be set to one of the child elements, or to the id of one of the child elements. Setting this property causes the slide to change.

slideshow.active = 'slide-1';

.autoplay

Boolean property. When true the slide-show activates the next slide after a pause. The pause duration may be set in CSS via the --slide-duration variable.

.controls

A TokenList object supporting the tokens "navigation", "pagination" and "fullscreen".

.loop

Boolean property. Makes the slideshow behave as a continuous loop.

Events

"slide-active"

Emitted by a slide when it is brought into scroll-snap alignment.

Parts

::part(prev-button)

Style the ‘previous slide’ button, available when the controls attribute contains the token "navigation".

::part(next-button)

Style the ‘next slide’ button, available when the controls attribute contains the token "navigation".

::part(pagination)

Style the pagination wrapper.

::part(page-button)

Style the pagination buttons.

::part(page-button-active)

Style the pagination button pointing to the active slide.

::part(fullscreen-button)

Style the fullscreen button, available when the attribute ‘controls’ contains the token "fullscreen".

::part(fullscreen-button-active)

Style the fullscreen button while in fullscreen mode.

Variables

--slide-duration

Set the duration that a slide is displayed for when the slideshow is in autoplay mode. May be a CSS time value in s or ms.

--padding-left

Adjusts the space to the left of the leftmost slide. Internally it sets padding and scroll-padding.

--padding-right

Adjusts the space to the right of the rightmost slide. Internally it sets padding and scroll-padding (and hacks around browsers’ lack of respect for padding-right on scrollable grids).
